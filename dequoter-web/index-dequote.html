Legend, Help, Terminology
term explanation
cust custom list i.e. a list of markers with irregular names separated by spaces
pat pattern list i.e. a list of markers with regular names that can be generated in the following format:
[marker name][optional separator][numbers from start to end, increasing by step amount]

start can be a bigger number than end, in which case step has to be a negative number
gen. table generate the quota table for the current definition
cust ↑↑ add a custom list to the start of the current quota table
cust ↓↓ add a custom list to the end of the current quota table
pat ↑↑ add a pattern list to the start of the current quota table
pat ↓↓ add a pattern list to the end of the current quota table
cust ↓ add a custom list right under the current list
pat ↓ add a pattern list right under the current list
